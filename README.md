## Getting Started

### Installation

1. Get a free API Key at [https://example.com](https://example.com)
2. Clone the repo
   ```sh
   git clone git@github.com:ChenShifu0308/LearnNewLanguanges.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run
   ```sh
   npm run dev
   ```
